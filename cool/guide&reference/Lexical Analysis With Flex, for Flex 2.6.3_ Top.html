flex

This manual describes flex, a tool for generating programs that perform pattern-matching on text. The manual includes both tutorial and reference sections.

This edition of The flex Manual documents flex version 2.6.3. It was last updated on 13 January 2017.

This manual was written by Vern Paxson, Will Estes and John Millaway.
